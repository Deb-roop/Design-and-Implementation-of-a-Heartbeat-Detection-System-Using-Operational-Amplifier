# Heartbeat Detection System Using Operational Amplifiers

## 📋 Project Introduction
This project was developed for the **Analog Electronics II Laboratory (ECE 2102) Open Ended Project** at **Khulna University of Engineering & Technology (KUET)**.

The human heartbeat is a critical physiological signal for monitoring health and detecting irregular heart activity early. This project focuses on designing and implementing a low-cost, portable heartbeat detector using operational amplifiers (Op-Amps). The system detects minute electrical or optical pulses from blood flow and amplifies them into a measurable form using analog signal conditioning, filtering, and amplification.

## 🎯 Objectives
* To design a functional heartbeat detector circuit using **LM358** Op-Amps.
* To amplify weak physiological signals and filter out unwanted environmental noise.
* To provide real-time visual heartbeat indication using LED or buzzer output.
* To gain practical experience in biomedical instrumentation and analog electronics.

## ⚙️ How It Works
1.  **Sensing**: An Infrared LED and Photodiode pair detect the pulse via blood flow in the human body.
2.  **Amplification**: The tiny signals from the photodiode are amplified by the **LM358 Op-Amp** to reach a usable voltage level.
3.  **Signal Conditioning**: Potentiometers and resistors control the gain and biasing to ensure the heartbeat is detected accurately while minimizing noise.
4.  **Output**: A red LED flashes in synchronization with each detected heartbeat, providing an immediate visual pulse rate.

## 🛠️ Main Components

| Item | Description | Function |
| :--- | :--- | :--- |
| **LM358 IC** | Operational Amplifier | Signal amplification and processing |
| **IR Pair** | IR LED & Photodiode | Heartbeat signal detection |
| **Passives** | Resistors & Potentiometers | Biasing, gain control, and filtering |
| **LED** | Red LED | Visual heartbeat indication |
| **Power** | 9V Battery | Portable power supply |

## 🌍 Social & Environmental Impact
* **Low-Cost Healthcare**: Provides an affordable solution for health monitoring in rural and remote areas where medical facilities are limited.
* **Sustainability**: Uses low-power analog components that are energy efficient and reusable for educational purposes.
* **Awareness**: Promotes personal health monitoring and preventive medicine.
